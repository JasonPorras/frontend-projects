## 🚀 Getting Started

Each project lives in its own folder and runs independently.  
Simply open the `index.html` file in your browser — no build step required.

```bash
git clone https://github.com/JasonPorras/frontend-projects.git
cd frontend-projects/todo-app
open index.html
```
